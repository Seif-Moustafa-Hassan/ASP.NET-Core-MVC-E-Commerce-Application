Clean Architecture: 3 Layers (3 projects in the same solution explorer)

Project 1: Main App (Controllers)
Project 2: Data (Models)
Project 3: Services (Interfaces and Services)

Roles: Admin - User
Permissions: View all Products - View Product Details - Create Product - Update Product - Delete Product - Add to Cart - View Cart

Admin: View all Products - View Product Details - Create Product - Update Product - Delete Product
User: View all Products - View Product Details - Add to Cart - View Cart
